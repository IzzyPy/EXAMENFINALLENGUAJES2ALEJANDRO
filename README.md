La aplicacion se llama Recetario, es una app en donde podemos crear, borrar actualizar y ver nuestras recetas, podemos realizar los crud en el
swagger y tambien tenemos una interfaz ui que se realizo con react css, que para verla tenemos que ejecutar la api primero, una vz ejecutada tenemos que ir a visual darle click derecho
a clientapp y abrir la carpeta en la terminal, escribir npm install, seguido de la instalacion escribir npm run dev, una vez hecho eso te dara una url localhost:5173 o 5174, en dond
al abrirla podras ver la interfaz ui para mejor comodidad y creacion de recetas.
